Why Image Dimensions Matter for Social Media

Every social media platform uses different image dimensions. Uploading a photo at the wrong size causes Instagram to compress and crop it unexpectedly, makes YouTube thumbnails appear blurry, and causes Facebook to add unwanted white borders. Using the correct pixel dimensions from the start ensures your photos look crisp and professional on every platform.

Social Media Image Size Guide (2024)

Here are the recommended image dimensions for every major platform — all available as one-click presets in this free resizer:

  • Instagram Post (square): 1080×1080 px — the most universally safe format for feed posts.
  • Instagram Story / Reel: 1080×1920 px — full-screen vertical format, also used for TikTok.
  • TikTok Photo / Video thumbnail: 1080×1920 px — always use vertical (9:16) for maximum screen coverage.
  • YouTube Thumbnail: 1280×720 px (HD 16:9) — the only size that renders correctly on all devices including TV.
  • Facebook Post: 1200×630 px — optimised for feed sharing and link previews.
  • Twitter / X Post: 1200×675 px — 16:9 landscape ratio for inline tweet images.
  • LinkedIn Post: 1200×627 px — recommended for articles, posts, and company updates.
  • Pinterest Pin: 1000×1500 px (2:3 ratio) — taller pins get significantly more impressions.
  • WhatsApp shared image: 800×800 px — compressed anyway, square format works universally.

JPEG vs PNG vs WebP — Which Format to Use?

JPEG is best for photographs — it produces the smallest file size with minimal visible quality loss. Use JPEG for Instagram, Facebook, and TikTok. PNG is lossless and best for images with text, logos, or transparent backgrounds. WebP offers the best compression of the three and is supported by all modern browsers and social platforms — use it when you need the smallest file without quality loss.
