About PhotoSplit Studio

A free, privacy-first browser tool that helps you split multiple scanned photos automatically — no uploads, no installation, no cost.

Our Mission

PhotoSplit Studio was created with one goal: to make digitizing a lifetime of printed photographs as easy and private as possible. Millions of families have shoeboxes full of old prints — memories from birthdays, holidays, and ordinary Tuesday afternoons that deserve to be preserved digitally. The problem has always been the tedious manual cropping step after scanning.

Traditional solutions either require expensive desktop software, upload your private family photos to someone else's server, or demand enough technical skill to make the process inaccessible for most people. PhotoSplit Studio eliminates all three barriers. It runs in your browser, it is completely free, and your photos never leave your device.

How It Works

PhotoSplit Studio uses a client-side computer vision algorithm built entirely in JavaScript and the HTML5 Canvas API. When you upload a scanned image, the tool analyses each row and column of pixels to find the high-contrast boundary between the white scanner background and the edges of each photograph. These detected regions are then outlined with interactive bounding boxes that you can adjust before saving.

Additional processing features — auto-straightening, color enhancement, and format conversion — are all performed using the Web Canvas API and Web Workers, keeping everything local to your device. The tool is distributed as a simple set of HTML, CSS, and JavaScript files that can even be saved and run offline.

Supported Features

  • Automatic multi-photo detection with adjustable sensitivity
  • Manual crop drawing for custom regions
  • Auto-straighten for de-skewing rotated photos
  • Auto color enhancement for restoring faded prints
  • JPEG, PNG, TIFF, BMP, WebP, and GIF input support
  • JPEG and PNG output with adjustable quality
  • Batch processing of multiple scans in one session
  • Full keyboard shortcut support for power users
  • Dark mode and responsive mobile layout

Privacy Commitment

We believe your family photographs are among the most personal data you own. PhotoSplit Studio was designed from the ground up to never touch that data. When you open an image file, it is read into your browser's local memory using the standard FileReader API. From that moment, all processing — detection algorithms, image transformations, file exports — happens exclusively inside your browser's JavaScript engine.

We have no back-end servers processing your images. We collect no analytics on which photos you scan. When you close or refresh the browser tab, all image data is immediately released from memory. Read our full Privacy Policy for complete details.
